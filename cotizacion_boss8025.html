Cotizacion de ajustes para BOSS8025

Base administrativa para areas, mantenimiento, limpieza, finanzas y reportes.

Cliente:
Osel Vega / Vegas Corporativo
Proyecto:
Web BOSS8025

Alcance del trabajo

Se trabajara sobre la web existente, aprovechando los modulos que ya estaban creados. La idea es dejar una base primaria mas solida para seguir agregando funciones en futuras etapas.

Concepto Que se hara
Areas Se cargaran y organizaran las etiquetas de areas de BOSS8025 para usarlas en mantenimiento, limpieza y reportes.
Mantenimiento Se ajustaran los formularios y vistas existentes para trabajar por area, descripcion y fecha final, conservando lo que ya tenia la web.
Limpieza Se mantendra el llenado actual para que siga siendo sencillo y se agregaran reportes semanal, quincenal, mensual y anual.
Finanzas Se ampliara el modulo para registrar departamento, mes, renta, pagos, gas, agua, luz, ingresos, egresos, totales y comparaciones por periodo.
Graficas y resumen Se agregaran totales de ingresos menos egresos y vistas de apoyo para revisar mes, ano y comparaciones con periodos anteriores.
Publicacion Se prepararan los cambios para subirlos al repositorio y aplicarlos en PythonAnywhere despues de probarlos.
